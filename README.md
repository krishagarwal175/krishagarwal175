##  Featured Work

###  Quadruped Robot

* Built from scratch with a **stability-first architecture**
* **PD control + IMU feedback loop** for real-time stabilization
* **Finite State Machine (FSM)** for gait control
* Hardware: Jetson Orin Nano, PCA9685, DS3240 servos
* Focus: making it **actually stable**, not just demo-ready

---

###  Vision & Perception Pipeline

* Depth estimation → image warping → masking → inpainting
* Working with models like Depth Anything + LaMa
* Goal: extract **usable spatial information**, not just visuals
